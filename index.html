What is Thingweb about?

Thingweb aims to be an exploratory implementation of both the WoT Model discussed in the W3C WoT IG and a RESTful HATEOAS based approach as discussed in the IRTF T2T PRG. It is simply a toolkit with some demo applications enabling you to create and experiment with WoT applications.
